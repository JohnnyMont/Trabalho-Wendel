Situação Problema
Um dos problemas gerais que pode ocorrer durante a configuração de cada modem, seria justamente alguma falha de configuração, um modo que foi configurado que o dispositivo não é adaptado para o modo desejado. As máquinas que se conectam nesses dois novos Access point após algum tempo ficam sem conectividade com a rede, se os usuários quiserem abrir ou salvar algum arquivo nas pastas da rede, por exemplo, (mapeamentos) não conseguem acessar, mostrando uma mensagem parecida com essa: "essa conexão foi perdida ou não foi restaurada", a conexão com a internet, e-mail continua funcionando.
Para resolver esse problema temporariamente é só pigar a máquina que você quer acessar e a máquina começa a enxergar novamente, mas apenas a máquina que você conectou. No modo Bridge o grande problema é que as maquinas só podem ter uma comunicação direta de dois para dois dispositivos. Já o modo repetição, eles só vão gerar um erro, caso há uma conexão falha devido o cabeamento correto, ou por falha interna na rede.
Como Resolver?
"Cada caso é um caso"
Dependendo do projeto pode optar por 3 tipos:
  • Modo AP(Access Point) ou Ponto de Acesso
  • Modo Bridge (Modo Ponte)
  • Modo Repetidor
Seguindo respectivamente, vamos falar de modo aprofundado as características de cada um.
Access Point
O Access Point é um modo do dispositivo que lhe permite de certa forma interligar duas redes wireless (sem fio) entre uma rede, vários dispositivos ou até mesmo um site. Em geral ele se conecta a uma rede cabeada e fornece uma conexão Wireless para dispositivos a uma determinada distância.
Bom, há várias formas de se configurar de até mesmo dispositivo access point. A configuração depende do serviço a ser prestado pelo access point, que nem sempre é um processo muito simples, pois os dispositivos são muitos versáteis, ou seja, oferecem vários modos de configuração para cada modo de configuração, e de certa forma fica até difícil de se escolher a configuração desejada.
Há várias outras maneiras de utilizar um access point, mas nem todos tem as mesmas possibilidades de interligação. Sempre verifique se o modelo a ser adquirido irá realmente atender às suas necessidades e se tem o modo de operação necessário a isto.
Bridge
O modo Bridge (Modo Ponte) quando um modem está na configurado para o modo bridge, ocorre uma conexão direta dos pacotes enviados para o computador conectado a rede já que há apenas uma rota e caminho da rede a ser seguido.
A analogia com uma ponte é bem interessante para explicar o funcionamento do modo bridge em dispositivos de rede. Um túnel também seria uma analogia interessante. O túnel ou a ponte (modo bridge) liga duas redes de comunicação entre si.
No caso do modem as duas partes que estão sendo ligadas não tem o mesmo protocolo de comunicação e nesta ligação o protocolo é convertido.
Repetidor
Já o modo Repetidor é a comunicação direta entre dois ou mais dispositivos encarregados de distribuir a rede compartilhada, e que um deles vão compartilhar a rede, como sendo a rede “primária”.
